# HTTP Server
Project 1 for COMP30023 Computer Systems 2018

---
## Project Detail
A simple HTTP server implements HTTP 1.0 and responds to GET requests

**Required MIME types**:

| Content | File Extension |
|:-:|:-:|
| HTML | .html |
| JPEG | .jpg |
| CSS | .css |
| JavaScript | .js |

---
## Usage
	make
	./server [port number] [path to web root]
	make clean
